# angular-ivy-hello-world
Card Element example code for Angular ivy

## Getting Started
* Clone repository
  ```
  git clone https://github.com/aelbore/angular-ivy-hello-world.git
  git checkout card
  ```
* Install dependencies
  ```
  npm install
  ```
* Build and Bundle
  ```
  npm run build
  ```
* Start the Application
  ```
  npm start
  browse: http://localhost:3000
  ```
